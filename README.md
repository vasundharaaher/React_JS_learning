# React JS Learning Journey 🚀

Welcome to my **React JS Learning Journey** repository.
This repository contains projects and notes created while learning React using different tooling setups.

---

## 📌 About This Repository

This repo demonstrates:

* Creating a React app using **Create React App (CRA)**
* Creating a React app using **Vite**
* Understanding the differences between both setups
* Managing dependencies using `npm`

The goal is to compare project structure, build tools, and development experience.

---

## 📁 Project Structure

```
React_JS_learning/
│
├── npm/
│   └── first-app/          # React app created using Create React App
│
└── vite-react-app/
    └── my-react-app/       # React app created using Vite
```

Each project contains its own `package.json` file and manages dependencies independently.

---

## ⚛️ 1️⃣ React App using Create React App

### 📂 Location

```
npm/first-app
```

### 🛠️ Created With

```
npx create-react-app first-app
```

### ▶️ Run the Application

```bash
cd npm/first-app
npm install
npm start
```

---

## ⚡ 2️⃣ React App using Vite

### 📂 Location

```
vite-react-app/my-react-app
```

### 🛠️ Created With

```
npm create vite@latest my-react-app
```

### ▶️ Run the Application

```bash
cd vite-react-app/my-react-app
npm install
npm run dev
```

---

## 🔎 Key Differences (CRA vs Vite)

| Feature          | Create React App  | Vite                   |
| ---------------- | ----------------- | ---------------------- |
| Build Tool       | Webpack           | ESBuild + Rollup       |
| Startup Speed    | Slower            | Very Fast              |
| HMR (Hot Reload) | Good              | Faster                 |
| Configuration    | Hidden by default | Lightweight & Flexible |
